# Project Watchlist

> Version 0.1.0

## Usage

Set the "config/config.env" values and change the NODE_ENV to either development or production.

## Install Dependencies

```
cd server && npm i
cd ../client && npm i
```

## Run Application

```
# Run in development mode
cd server && npm run dev

# Run in production mode
cd server && npm start
```
